# Copy Trader Monitor

The Copy Trader Monitor tab is the primary interface for tracking the performance and activity of the PolyEdge copy trading engine. It allows you to monitor the whales and expert wallets the bot is currently mirroring.

## Access
Access to this monitor requires admin login. The copy trading strategy must be enabled in the Strategies tab for data to appear here.

## Strategy Status
The top header indicates if the copy trading engine is **Enabled** (Green) or **Disabled** (Gray). It also displays the total number of tracked wallets across all supported prediction markets.

## Tracked Wallets
A real-time table lists all wallets currently being monitored for copy trading opportunities:

| Column | Description |
|--------|-------------|
| **Wallet** | The 0x address of the tracked wallet (e.g., Polymarket whale). |
| **Pseudonym** | A user-friendly name for the wallet (if configured). |
| **Score** | A proprietary ranking score based on historical P&L and win rate. |
| **30d Profit** | Total realized profit or loss for the wallet over the last 30 days. |

:::tip
Profit values are color-coded: **Green** for positive profit and **Red** for losses.
:::

## Recent Signals
The **Recent Copy Signals** section displays a feed of recent trade decisions generated by the copy trading engine:

| Field | Description |
|-------|-------------|
| **Market** | The ticker symbol or ID of the prediction market. |
| **Decision** | The trade action taken: **BUY** (Green) or **SELL** (Red). |
| **Confidence** | The bot's confidence level in the copied signal (0-100%). |
| **Time** | The timestamp of when the signal was generated. |

## How It Works
The copy trading engine polls the Polymarket and Kalshi APIs for recent activity from a curated list of top-performing wallets.
- **Wallet Selection**: High-scoring wallets are automatically added to the track list based on their historical P&L and win rate.
- **Signal Generation**: When a tracked wallet enters a position, the bot generates a copy signal. The signal's confidence is weighted by the wallet's score and the current market edge.
- **Execution**: If the signal's confidence exceeds the threshold (defined in Settings), the bot executes a mirrored trade using the configured Kelly fraction.

:::warning
Copy trading involves risks, including following a whale who may be hedging or entering a complex position. Review recent signals carefully before enabling fully automated copy trading.
:::

:::tip
You can manually add or remove tracked wallets in the **Wallet Config** tab.
:::
