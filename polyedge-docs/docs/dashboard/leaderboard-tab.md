# Leaderboard Tab

The Leaderboard tab provides a detailed view of the top-performing traders on Polymarket. It allows you to track their performance, evaluate their strategies, and even copy their trades.

## What You'll See

This tab is organized into:
1. **Search + Filter Row**: Controls to find specific traders or groups of traders.
2. **Leaderboard Table**: A ranked list of the most successful traders based on profit, win rate, and total trades.

## Filters

| Filter | Options | Description |
|--------|---------|-------------|
| **Search** | Text | Find a trader by their wallet address or pseudonym. |
| **Profit** | Number | Filter traders with a 30-day profit greater than or equal to a specific amount. |
| **WR** | Percentage | Filter traders with a win rate greater than or equal to a specific percentage. |
| **Trades** | Number | Filter traders with a total number of trades greater than or equal to a specific amount. |

## Understanding the Data

### Leaderboard Columns

| Column | Description |
|--------|-------------|
| **Rank** | The trader's position on the leaderboard (e.g., `#1`, `#2`). |
| **Trader** | The trader's pseudonym or a truncated wallet address. |
| **Profit** | The trader's realized profit over the last 30 days. |
| **Win Rate** | The percentage of successful trades for this trader. |
| **Trades** | The total number of trades made by this trader. |
| **Markets** | The number of unique markets this trader has participated in. |
| **Score** | A composite score used to rank the trader's performance. |

## Controls

- **Sort**: Click on column headers to sort the leaderboard by profit, win rate, total trades, or score.
- **Copy**: Click the `Copy` button to add a trader's wallet address to the Copy Trader strategy. This allows the bot to monitor and replicate their positions in your own account.

## Tips

:::tip
Sort by **Profit** to find the most successful traders, or sort by **Win Rate** to find the most consistent ones.
:::

:::info
The leaderboard data refreshes every 60 seconds to include the latest performance metrics for top traders.
:::

:::warning
Copying trades involves significant risk. Always evaluate a trader's history and risk profile before enabling the Copy Trader strategy.
:::
